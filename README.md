# RubinTracker

Plataforma SaaS para jogadores do RubinOT acompanharem sua evolução.

Stack inicial:
- Next.js 15 (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn/UI
- Lucide Icons
- Recharts
- React Hook Form + Zod
- Prisma + PostgreSQL
- Supabase Authentication

Escopo deste commit: scaffold inicial do projeto e arquitetura de pastas. Próximos passos: configurar infra, criar telas e flows de autenticação, modelagem detalhada do banco.
